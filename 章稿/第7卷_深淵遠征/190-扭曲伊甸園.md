# 第190章 扭曲伊甸園

有了老喬的權限卡，溫室後方的門順利打開。

但門後的景象，卻讓所有人都倒吸一口涼氣。

這裡不再是花園，而是一個……屠宰場。

準確地說，是植物與幾何體的戰場。

地面上鋪滿了厚厚的黑色粘液，那是幾何生物崩解後的殘骸。而在粘液中，生長著無數扭曲的、長滿利齒的藤蔓。它們像是有意識的蛇群，正在瘋狂地吞噬著那些幾何碎片。

「這就是『扭曲伊甸園』。」

灰醫看著眼前的一幕，語氣沉重。

「當年實驗室失控後，這裡的植物為了生存，主動吸收了幾何輻射。它們從生產者變成了消費者。甚至……獵食者。」

「小心！」

秦嶺突然拔槍，對著頭頂就是一槍。

「嘶——」

一條半透明的、如同水晶一般的藤蔓從天花板垂下來，剛好被子彈打斷。斷口處噴出的不是汁液，而是強酸。

「這種酸能腐蝕機甲。」秦嶺警告道。

「這裡簡直就是個生化地獄。」老巴看著那些還在蠕動的斷藤，覺得頭皮發麻，「我們怎麼過去？」

「飛過去。」李塵看了一眼韓昭。

韓昭會意，雙手結印。

「空間屏障！起！」

一個透明的球形空間護盾將眾人包裹起來，緩緩升空。

但就在這時，下方的植物似乎感應到了獵物的這不是它們能吃的東西，而是……入侵者。

轟！

無數藤蔓沖天而起，像是一張巨大的網，試圖攔截空中的護球。

「不知死活！」老巴機炮掃射。

但在這裡，物理攻擊的效果大打折扣。那些藤蔓被打斷後，幾秒鐘就能重新長出來。而且它們還會釋放出大量的麻痺毒霧。

「屏障撐不住多久！」韓昭面色蒼白，「毒霧正在腐蝕空間壁！」

「用火！」李塵喊道。

「這裡要是起火，我們這些氧氣含量，會瞬間爆炸！」灰醫阻止道。

「那就用冰！」

李塵手中的骨鑰再次發光。這一次，他調動的是從【深淵冰魄】（早期副本獲得的道具）中提取的寒冰規則。

「絕對零度！」

咔嚓！咔嚓！

一股極寒的氣息瞬間擴散。那些張牙舞爪的藤蔓在接觸到寒氣的瞬間，變成了晶瑩剔透的冰雕。

然後，碎裂。

「趁現在！衝過去！」

韓昭咬牙，空間球加速衝刺，終於在屏障崩潰的最後一秒，衝進了對面的一條走廊。

「呼……」

眾人落地，驚魂未定。

這裡已經是核心區的外圍了。

走廊的兩側，排列著無數個巨大的玻璃罐。罐子裡充滿了綠色的液體，而在液體中，漂浮著各種奇形怪狀的……生物。

有的長著三個頭，有的沒有皮膚，有的乾脆就是一團肉塊。

「這些……都是適格者？」阿岬走到一個罐子前，看著裡面漂浮的一個只有半邊身體的胚胎，眼中滿是恐懼。

「失敗品。」灰醫走過來，眼神中充滿了痛苦，「或者是被幾何規則侵蝕後，發生畸變的實驗體。我們當年……真的做了很多孽。」

秦嶺卻沒有停留。他徑直走到走廊盡頭的一個巨大金屬櫃前。

這個櫃子很特別。它不是為了保存標本，而是為了……封印。

上面貼著黃色的警示條：

**【絕密檔案 - 編號 12】**
**【危險等級：S】**
**【狀態：封存】**

「這是你的檔案？」李塵走了過來。

秦嶺點點頭。他的手放在櫃門上，卻遲遲沒有打開。

「其實……我一直都知道。」秦嶺低聲說道，「我的記憶是被修改過的。我知道自己在黑潮是一個完美的戰士，一個沒有感情的殺手。但我不知道，為什麼我會變成這樣。」

「也許……裡面有你想知道的答案。」

秦嶺深吸一口氣，猛地拉開櫃門。

裡面沒有什麼驚天動地的秘密武器。只有一份薄薄的紙質文件，和一段錄音筆。

秦嶺顫抖著手，拿起文件。

第一頁，就是一張照片。

照片上，是一個只有五六歲的小男孩，正抱著一隻瘸腿的小狗笑得燦爛。而在他身邊，站著一個年輕的男人，正溫柔地摸著他的頭。

那個男人……是灰醫。

「不……」秦嶺後退了一步，難以置信地看著灰醫，「這不可能……我不認識你……」

「你認識。」灰醫的聲音有些沙啞，「你叫秦小天。你是……我的兒子。」

轟！

這句話像是一道驚雷，狠狠地劈在秦嶺的腦海裡。

無數被封印的記憶碎片在這一刻決堤而出。

陽光、草地、小狗……父親溫暖的大手……還有那一天，母親被黑星輻射變異成怪物的慘叫……父親為了救他，不得不將他放入實驗艙，切除痛覺神經，改造成「最強兵器」來抵抗侵蝕……

「啊——！！！」

秦嶺跪在地上，抱著頭發出痛苦的嘶吼。

那被切除了十年的痛覺，在這一刻，彷彿以一種靈魂層面的方式回歸了。這比肉體的疼痛要強烈一千倍，一萬倍。

「你騙我……黑潮騙我……」

秦嶺的眼中流下了血淚。

原來，那個沒有感情的殺人機器，曾經擁有了世界上最溫暖的愛。而親手剝奪這一切的，正是那個聲稱要保護他的組織（黑潮後來接管了實驗室，篡改了記憶）。

「對不起，小天。」灰醫走過去，想要抱住他，「爸爸當時……沒有別的選擇。」

「別過來！」秦嶺猛地抬頭，手中的槍指著灰醫，「你為什麼不早說！為什麼要讓我不人不鬼地活著！」

「因為只有這樣，你才能在那個吃人的黑潮裡活下來。」灰醫閉上眼睛，任由槍口指著自己，「如果讓他們知道你還有情感，還有弱點，你早就被銷毀了。」

李塵在旁邊嘆了口氣。

這真是一個操蛋的世界。父子相殘，為了生存不得不變成怪物。

但他沒有插手。這是他們的家務事，必須由他們自己解決。

良久。

秦嶺手中的槍慢慢放了下來。

他擦掉眼角的血淚，重新站了起來。雖然臉色依然蒼白，但眼神中卻多了一種前所未有的東西。

那是人性。

「黑潮……」秦嶺咬著牙，每一個字都像是從牙縫裡擠出來的，「這筆賬，我記下了。」

他看向走廊盡頭那扇沉重的大門。

「裡面就是 0 號大腦。也就是……這一切悲劇的源頭。」

秦嶺換上了新的彈匣，眼神變得無比堅定。

「爸爸。」

他雖然沒有看灰醫，但這聲稱呼，卻讓灰醫老淚縱橫。

「我們去把那個破電腦砸了。然後……回家。」

「好。」灰醫擦乾眼淚，「回家。」

李塵看著這對重新相認的父子，嘴角勾起一抹微笑。

「那麼，斬首小隊，全員集結。」

他舉起骨鑰，指向那扇最後的大門。

「目標：0 號大腦。行動代號：回家。」

（本章完）
