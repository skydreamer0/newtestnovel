# 第191章 0 號大腦

推開那扇標記著「核心禁區」的沉重鉛門，一股刺骨的寒意撲面而來。

這不是溫度的冷，而是某種直接作用於精神層面的「絕對理智」。這裡沒有任何裝飾，沒有任何多餘的線條。整個空間就像是一個巨大的、完美的球體內部。

而在球體的正中央，懸浮著一個直徑超過十米的巨大透明容器。

容器裡充滿了半透明的藍色營養液，而在液體中，漂浮著一個……大腦。

一個巨大的、還在微微搏動的大腦。

它不像是人類的大腦，因為它的表面覆蓋著一層銀白色的金屬網，無數光纖像神經一樣插入它的溝回之中，發出有節奏的藍光。

「這就是……0 號。」

灰醫抬起頭，看著這個曾經由他親手設計、如今卻已經完全陌生了的造物。

「它是帝國最強大的生物計算機。也是『適格者計畫』的母體。阿岬、秦嶺，甚至包括 T-001，它們的底層基因代碼和邏輯算法，都是從這裡衍生出來的。」

李塵也開啟了【全知視角】。

但在他的視野中，這個大腦是一團巨大的、混亂的數據風暴。

**【警告！檢測到超高強度精神輻射！】**
**【警告！目標正在嘗試建立神經鏈接……】**

「誰……在那裡……」

一個恢弘的聲音在眾人腦海中響起。不像 T-001 那種冰冷的電子音，這個聲音聽起來很蒼老，很疲憊，像是一個在睡夢中被驚醒的老人。

「我是 0 號……我是……真理的容器……」

隨著聲音響起，那個巨大容器裡的營養液開始沸騰。無數氣泡升騰，大腦表面的光芒驟然變亮。

「啊！」

阿岬突然抱住頭，發出痛苦的呻吟。

「它在叫我……它在叫我回去……」

不僅是阿岬，秦嶺的身體也開始顫抖。他體內的納米修復單元似乎失控了，瘋狂地運轉著，試圖強制接管他的運動神經。

「全體退後！」李塵大喝一聲，精神力爆發，形成一個護盾將兩人罩住，「這傢伙在試圖『格式化』你們！」

「不……不是格式化……」灰醫咬著牙，臉色蒼白，「是『歸巢』。它認為我們是它丟失的數據包，想要把我們回收。」

「那就讓它閉嘴！」

李塵舉起骨鑰，想要像對付 T-001 那樣強行鎮壓。

但這一次，骨鑰的光芒剛一亮起，那個巨大的大腦就有了反應。

「白燼……古拉的遺骨……」

0 號大腦的聲音突然變得激動起來。

「鑰匙……終於來了……解鎖……解鎖……」

轟！

一道無形的精神衝擊波猛地爆發。

李塵只覺得眼前一黑，整個人像是被捲入了一個巨大的漩渦。周圍的物理世界消失了，取而代之的，是一個由無數數據流組成的純白空間。

這是……深層神經網絡？

李塵發現自己懸浮在這個白色空間裡，而在他對面，站著一個老人。

那個老人和灰醫長得一模一樣，但更年輕，眼神更加銳利，也更加狂熱。他穿著一身潔白的實驗服，胸口掛著「首席科學家」銘牌。

「歡迎來到我的思維殿堂，繼任者。」年輕版灰醫微笑著說道。

「你是 0 號？」李塵警惕地問。

「我是 0 號的人格面具。也就是當年首席留在這裡的『理想自我』。」老人推了推眼鏡，「外面的那個老頭子，是他懦弱的『現實自我』。」

「有意思。」李塵冷笑，「一個精神分裂的電腦。」

「可以這麼說。」老人並不生氣，「這三百年來，我一直在計算一個問題：人類如何才能在黑星的規則下生存？最後，我得出了一個結論。」

他一揮手。

周圍的空間瞬間變換。出現了無數個畫面。

有長滿晶體的人類在廢墟中遊蕩；有完全機械化的人類在地下建立蜂巢；有變成能量體的意識在網絡中穿梭。

「融合。」老人說道，「只有放棄脆弱的肉體，與規則融合，才能獲得永恆。」

「就像 T-001 那樣？」李塵反問。

「T-001 是個失敗品。它太極端了，只追求幾何的完美，卻丟棄了靈魂。」老人搖搖頭，「而我，找到了更完美的方案。」

他突然指向李塵。

「那就是你。」

「我？」

「是的。你擁有神格碎片，你擁有修改規則的能力，但你依然保持著人類的情感和靈魂。你就是那個『完美的容器』。」

老人的眼神變得狂熱。

「把你的身體交給我。把骨鑰交給我。我將帶領人類完成最後的進化！」

無數白色的數據鎖鏈從虛空中探出，纏向李塵的靈魂。

「又是一個想奪舍的。」

李塵嘆了口氣。

「你們這些搞科研的，是不是都有這個毛病？總覺得自己比上帝還聰明。」

金色的火焰在他身上燃燒起來。

那不是普通的火，那是【鍊成之火】。

「在我的規則裡，我才是上帝。」

李塵一拳轟出。

金色的拳印帶著摧枯拉朽的氣勢，狠狠地砸在老人的臉上。

「這一拳，是替外面的那個老頭子打的！」

「轟！」

老人的臉被打碎了，化作無數數據碎片。

「不可能……你的權限……」

「我的權限？」李塵冷笑，「我是你們這個世界的『管理員』，懂嗎？」

他伸出手，虛空一抓。

【系統指令：強制重啟。】
【權限來源：最高管理員（偽）。】

整個白色空間開始崩塌。

現實世界中。

「噗！」

容器裡的 0 號大腦猛地收縮了一下，藍色的光芒迅速黯淡下去。

李塵猛地睜開眼，大口喘著粗氣，鼻血滴落在地上。

「老闆！」老巴衝過來扶住他。

「我沒事。」李塵擦掉鼻血，看著前方，「搞定它了。」

容器裡的液體平靜了下來。那個巨大的大腦不再發出精神衝擊，而是溫順地漂浮著，像是一個等待指令的巨大硬碟。

灰醫顫顫巍巍地走到控制台前。

「它……休眠了。」

「現在你是新的管理員了。」李塵說道，「去吧，拿到我們要的東西。蘇芮的解藥。」

灰醫點點頭，雙手放在鍵盤上。

屏幕上瘋狂地刷過無數代碼。

幾分鐘後。

「找到了。」灰醫的聲音帶著一絲顫抖，「『頻率穩定劑』的配方。還有……」

他調出了一段視頻日誌。

視頻裡，年輕的首席科學家對著鏡頭，神色絕望而決絕。

「……這是一場騙局。帝國高層早就知道黑星的真相。他們不是在抵抗，而是在『獻祭』。他們想用整個星球的生命能量，去換取那艘逃生飛船的啟動……」

「我不能讓他們得逞。我將封鎖實驗室，帶走『鑰匙』。如果有人看到這段影像，請記住……」

「深淵的對面，不是毀滅，而是……鏡子。」

視頻戛然而止。

眾人一片死寂。

「鏡子？」阿岬喃喃自語，「是什麼意思？」

「不知道。」李塵皺眉。這個信息量太大了，似乎觸及到了黑星的本質。

但現在沒時間思考哲學問題了。

「配方既然拿到了，馬上合成！」李塵下令，「我們還剩多少時間？」

灰醫看了一眼進度條。

「合成需要 12 小時。而且……需要巨大的能量支持。」

「用 0 號大腦的備用能源呢？」

「可以。但是……」灰醫欲言又止，「啟動合成程序的能量波動極大，會像燈塔一樣照亮整個地下深淵。」

「你是說……」

「那些被我們趕走的幾何生物，會像飛蛾撲火一樣湧過來。」

李塵深吸一口氣，握緊了手中的槍。

「那就讓它們來。」

他轉身看著隊友們。

老巴正在架設機炮，秦嶺正在擦拭短刀，阿岬正在閉目養神。

「12 小時。」李塵沉聲說道，「我們要在這裡，死守 12 小時。」

「守得住嗎？」老巴問了一句大實話。

「守不住也要守。」李塵看向那個巨大的大腦容器，目光透過它，彷彿看到了遠在 T-17 塔底沉睡的蘇芮。

「為了女王。」

「為了回家。」秦嶺接了一句。

「開始吧。」

灰醫按下了「確認」鍵。

嗡——

0 號大腦再次亮起，但這次是柔和的綠光。巨大的能量開始運轉，合成程序啟動。

與此同時。

深淵的黑暗中，無數雙猩紅、幽藍的眼睛，再次睜開了。

（本章完）
