# 神性主角提升與架構優化（對照第1-45章）

## 目的
把「神性成長爽感」和「制度博弈張力」同時拉高，避免後段只剩程序戰或只剩戰鬥戰。

---

## 一、神性類型小說常見主角提升框架

### 1) 六條升級軸（建議固定追蹤）
1. 神性量：信仰來源與輸出上限是否擴大。
2. 神性質：信仰品質是否從本能走向自主追隨。
3. 載體耐受：裂痕、反噬、污染是否被管理而非硬扛。
4. 權柄合法性：是否拿到制度授權，能「合法使用力量」。
5. 組織化能力：是否能把個人戰力變成可複製流程。
6. 情感代價：每次變強是否伴隨關係、身體或選擇成本。

### 2) 常見失速點（需避免）
1. 只堆數值，不改規則地位。
2. 反派升級慢於主角，戰鬥變單向碾壓。
3. 代價只寫痛苦，不改行動選擇。
4. 世界觀只補資料，不落地到交易、票據、流程。

---

## 二、你目前版本（第1-45章）評估

### 已經很強的部分
1. 成長飛輪清楚：修復 -> 量產 -> 奪點 -> 神性 -> 牌桌。
2. 代價有落地：裂痕、封約、污染、抽樣、斷供都在吃主角行動空間。
3. 勢力升級有層次：從棚區幫派到灰環程序，再到井口權柄。
4. 章節節奏穩定：大多能做到「壓力 -> 操作 -> 回收」。

### 可優化處（關鍵）
1. 神性進度缺統一儀表：目前有裂痕與反噬，但缺固定三欄指標（神性輸出、裂痕回流、合法授權）。
2. 人類信仰線偏慢：鼠群信仰已成熟，但「人類追隨信仰」尚未成為主要增幅來源。
3. 反派線可再收束：牽魂師殘線、教團、黑潮、特管組都在場，需明確每10章主壓力來源。
4. 程序戰占比漸高：需保留制度爽點，但要綁定實體收益（貨權%、窗口時長、運價、保費）。

## 三、架構調整建議（可直接套用）

### A. 每章固定雙線，禁止單線空轉
1. 行動線：戰鬥、潛入、護航、撤離至少一項。
2. 規則線：條款、驗簽、證據鏈、席位變化至少一項。

### B. 每3章固定一次「神性抉擇」
1. 開高輸出拿短期勝利，代價是裂痕上跳。
2. 保守輸出保長期穩定，代價是窗口流失或盟友受損。

### C. 每10章固定一次「規則地位升級」
1. 不是只打贏人，而是拿到新規則按鈕。
2. 範例：從正式席 -> 驗簽席 -> 分層貨權席。

### D. 每次大勝都要有「可見帳單」
1. 身體帳：裂痕、灼痕、失能時長。
2. 組織帳：隊伍傷耗、供應損耗、聯盟票數。
3. 制度帳：條款限制、審核升級、保證金提高。
