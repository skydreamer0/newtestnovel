# 能力與修煉

## 金手指雙核心

### 廢界鍊成爐（物質）
- 能力：分解、提純、修復、重組。
- 重點優勢：可顯示目標物「缺件清單」與最近補件來源。

### 魂譜刻印（靈魂）
- 能力：讀魂、微調本能、模板刻印。
- 主要用途：把鼠群做成可協同運作的功能編隊。

## 資源循環（主角成長引擎）
1. 掃描垃圾 -> 判定價值。
2. 分解低值廢料 -> 拿基礎材料。
3. 補齊缺件 -> 修復高值設備。
4. 把利潤投入工坊與鼠群擴編。
5. 用鼠群搶更高級投放區與神性材料。

## 鼠群編隊
- 食腐鼠：資源回收。
- 哨戒鼠：偵查、追蹤、警戒。
- 工匠鼠：工坊搬運和微組裝。
- 噬靈鼠：戰鬥與靈魂干擾。
- 祈禱鼠：維持信仰回路與神性輸出。

## 修煉體系（神格路線）
1. 見習刻印者：改單體。
2. 牧魂者：改群體。
3. 權柄執事：建區域回路。
4. 神性主教：凝權柄雛形。
5. 半神代行：改高智慧生命。
6. 星域神座：建立個人神域。

## 平衡限制（避免一瞬爆棚）
1. 不能無中生有，只能重組既有材料。
2. 鍊成負荷值每日上限固定，超載會重傷。
3. 產線槽位有限，擴產要升級工坊。
4. 材料品質決定成品上限，不能越級白嫖。
5. 魂印污染會導致鼠群失控。
6. 神格融合有排異與失敗率。

## 可持續爽點模板
- 小爽：補齊缺件，立刻修好值錢裝備。
- 中爽：鼠群分工成功，低成本打贏高成本敵人。
- 大爽：奪據點 + 拿到稀有神性材料 + 當場突破。
