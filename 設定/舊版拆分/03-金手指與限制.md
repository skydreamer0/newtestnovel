# 金手指與限制

## 雙核心系統

### 廢界鍊成爐（物質）
- 分解垃圾為基礎資源。
- 修復殘件與古代設備。
- 重組裝備、生物培養模組。
- 顯示缺件清單與最短補件路徑。

### 魂譜刻印（靈魂）
- 讀取靈魂屬性。
- 微調本能與傾向。
- 建立鼠群模板化魂印。
- 形成分工群體行為。

## 爽點流程
撿到破爛 -> 系統判定可修復 -> 提示缺件 -> 補件成功 -> 裝備升級 -> 鼠群擴編 -> 戰力暴增。

## 限制機制（防爆棚）
1. 不能無中生有，必須先有材料。
2. 每日鍊成負荷值固定，超載會昏迷或靈魂灼傷。
3. 產線槽位有限，同時修復與製造數量受工坊等級限制。
4. 品質鎖存在，低階材料無法直升高階神器。
5. 魂印污染會導致鼠群異化或失控。
6. 神格排異存在失敗率與副作用。
