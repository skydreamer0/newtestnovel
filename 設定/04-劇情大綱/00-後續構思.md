# 長篇宏觀規劃（千章版）

為支撐 **1000章+** 的量級，故事結構不能僅停留在「垃圾場星域」。世界觀必須進行三次「地圖升級」（Map Expansion），每一層地圖對應一套全新的資源、敵人和社會規則。

## 核心節奏（Saga Structure）

- **第一部：廢土霸主篇（Ch 1 - 250）**
  - **地圖**：垃圾場星域（廢星 + 近軌空間 + 停泊環）
  - **核心**：從求生到統一。建立T-17帝國，徹底征服黑潮與聖骸，成為星域之王。
  - **金手指階段**：內化 -> 點火 -> 燃料積累。

- **第二部：深淵遠征篇（Ch 251 - 550）**
  - **地圖**：零號墜神井內部（古戰場 + 摺疊空間 + 異種文明）
  - **核心**：發現「垃圾」的來源。與深淵原住民（遺民/異種）博弈，還原古神戰爭真相。
  - **金手指階段**：熔解 -> 分離（物理規則修改）。

- **第三部：諸神黃昏篇（Ch 551 - 850）**
  - **地圖**：舊神疆域（高維碎片世界 + 神國殘骸）
  - **核心**：屠神證道。主動獵殺甦醒的舊神，奪取完整權柄，將靈魂海升格為神國。
  - **金手指階段**：提純 -> 結晶（概念規則修改）。

- **第四部：萬物重鑄篇（Ch 851 - 1000+）**
  - **地圖**：外域/宇宙真理層（對抗熵增/外神）
  - **核心**：面對真正毀滅古文明的元兇。以鍊成爐重寫宇宙法則，定義新世界。
  - **金手指階段**：重鑄（創世）。

## 詳細卷目規劃

### 第一部：廢土霸主（Ch 1 - 250）
*目標：將T-17從一個塔變成一個國。*

- **第一卷：源代碼重啟（Ch 1-30）** `[已完結]`
  - 生存與金手指獲得。
- **第二卷：井口權柄（Ch 31-60）** `[已完結]`
  - **核心**：奪取九號井權柄，建立裂羽聯航。
- **第三卷：聯盟崩塌（Ch 61-90）**
  - **核心**：聖骸教團擴張，舊秩序崩潰，聯航正式獨立。
- **第四卷：星域一統（Ch 91-250）**
  - **核心**：總決戰與地圖統一。
  - **核心**：總決戰。主角整合所有中小勢力，對剩餘巨頭進行圍剿。
  - **結局**：主角加冕為「垃圾場之王」，整合全星域資源，準備打開通往深淵的大門（零號井）。

### 第二部：深淵遠征（Ch 251 - 550）
*目標：從井口跳下去，看看神是怎麼死的。*

- **第五卷：墜落地（Ch 251-320）**
  - **新地圖**：深淵第一層「骸骨平原」。這裡的物理規則與外面不同（重力異常/時間流速不同）。
  - **衝突**：遭遇第一批「深淵遺民」（當年沒撤走的古文明後裔），發生文化與科技衝突。
- **第六卷：失落科技樹（Ch 321-400）**
  - **核心**：尋找古文明的兵工廠。主角發現現有的神術其實是古科技的劣化版。
  - **高潮**：重啟「泰坦級」生產線，T-17 科技躍遷。
- **第七卷：異種戰爭（Ch 401-480）**
  - **衝突**：深淵深處的「噬神種」（以神格為食的生物）甦醒，湧向地面。主角必須在「保衛地面」和「深入地下」中做選擇。
- **第八卷：深淵之心（Ch 481-550）**
  - **核心**：抵達深淵最底層。發現「神格」其實是古神試圖封印「外域污染」的容器。
  - **結局**：主角觸碰深淵之心，靈魂海徹底破碎重組，晉升半神。

### 第三部：諸神黃昏（Ch 551 - 850）
*目標：獵人變獵物，主角開始吃神。*

- **第九卷：神國復甦（Ch 551-620）**
  - **事件**：因為深淵之心的變動，舊神們開始在各個維度甦醒。
  - **衝突**：舊神試圖收回權柄，星域人類淪為獻祭品。
- **第十卷：弒神者（Ch 621-720）**
  - **核心**：主角利用鍊成爐「分離」特性，剝離神的權柄。這是一場不對稱戰爭（人 vs 規則）。
  - **高潮**：第一場弒神戰。主角將一尊火神鍊成為「永恆能源核心」。
- **第十一卷：絕對王座（Ch 721-850）**
  - **核心**：主角建立神系。將T-17的老夥伴們（阿岬、蘇芮等）升格為從神/天使。
  - **結局**：肅清星域內所有舊神勢力，主角成為唯一真神。

### 第四部：萬物重鑄（Ch 851 - 1000+）
*目標：面對真正的恐懼——外域。*

- **第十二卷：天外來客（Ch 851-920）**
  - **揭秘**：外域掠奪者（當初毀滅古文明的元兇）感應到新神誕生，再次降臨。
  - **絕望感**：對手是跨宇宙級別的文明，神力剛起步的主角方陷入苦戰。
- **第十三卷：最後的鍊成（Ch 921-1000）**
  - **核心**：不僅是戰鬥，而是規則比拼。
  - **高潮**：主角發動鍊成爐最終權限「萬物重鑄」。獻祭整個舊神系（或自身神格），將星域重寫為一個不可被入侵的獨立維度。
- **終章：新世界（Ch 1000+）**
  - 戰後的新世界面貌。回歸初心（或許主角又變回了一個普通的維修師，但在一個完美的世界裡）。

## 擴充對策總結
1. **拉長第二卷**：將原定的「井口權柄」拆解，本卷只專注於「商業戰爭與初步崛起」，不急著統一。
2. **增加地圖**：不只是「深淵」，還有「深淵內部的多層世界」和「高維空間」。
3. **戰力通膨控制**：
   - 1-250章：槍砲與錢。
   - 251-550章：古科技與異能。
   - 551-1000章：規則與概念。
