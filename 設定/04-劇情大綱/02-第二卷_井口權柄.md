# 第二卷：井口權柄

## 卷核心一句話
主角帶著第一枚神格憑證進入黑潮牌桌，爭奪零號墜神井井口權柄，從守塔者升級為星域供應鏈玩家。

## 第二卷五大目標（不變）
1. 取得零號墜神井「分層貨權」至少一層，打開外域資源入口。
2. 建立T-17到灰環停泊場的安全運輸鏈，不再被單方斷供。
3. 找到修補魂譜裂痕與母巢活性的方法，避免主角戰力短命化。
4. 逼出黑潮與聖骸的真實上下游關係，拆掉其中一條控制鏈。
5. 拿到第二枚神格碎片線索，形成「1/3 -> 2/3」的卷末拉升。

## 已落地進度（第31-45章）

| 區段 | 已完成內容 | 劇情結果 |
|---|---|---|
| 第31-35章 | 灰環赴約 -> 牌桌破局 -> 臨時試運 -> 首航截擊 -> 反打航道 | 序號13保住並升正式席，拿到裂羽-乙線維護優先權 |
| 第36-40章 | 井口試探 -> 夜拍黑標 -> 外環踏井 -> 封線追殺 -> 樞紐奪權 | 鎖定雙鎖結構，拆掉清洗隊單方封線，建立聯航驗簽 |
| 第41-45章 | 熄神井道 -> 白燼骨鑰 -> 井口審衡 -> 封約反噬 -> 盲井聽壁 | 拿到骨鑰母鍵序列63%，保住進井資格，取得A級深層資料 |

## 後半節奏骨架（第46-60章修訂）
1. 第46-48章：把A級深層資料轉成「可執行進井路徑」，拿到下一輪穩定窗口。
2. 第49-51章：聯航內部整編，人類信仰線入場，主角從「被護航」變成「能護航他人」。
3. 第52-54章：裂痕修補與穩定材供應鏈落地，神格輸出從爆發型走向持久型。
4. 第55-57章：井口雙鎖總對沖，正面爭奪分層貨權席位。
5. 第58-60章：卷末對賭，鎖定第二枚神格碎片線索並完成 1/3 -> 2/3 拉升。

## 升級與衝突同步規則
- 戰場升級：單塔守城 -> 航線路由 -> 井口配額與封條雙鎖。
- 反派升級：地方審判線 -> 特管清洗線 -> 黑潮高層與教團合流。
- 爽點升級：修復撿漏 -> 奪航權 -> 改規則 -> 爭井口分層貨權。
- 代價升級：裂痕惡化、封約反噬、供應鏈成本、聯盟票數風險同步上升。

## 第二卷寫作守則補充（後半）
- 保持「每3章一小高潮、每10章一大高潮」節奏，不改。
- 權謀章不可空談，必須落到可量化收益：貨權%、運價、供應量、窗口時長至少一項。
- 每章至少推進兩條線：行動線（打/潛/運）與規則線（驗簽/條款/證據）。
- 主角每次大勝後必帶一項可見代價，且代價需影響下一章決策。

## 第46-50章細綱（對齊新版前五章規則）

### 先行硬規則（避免後續寫歪）
- 金手指已在第3章完成內化，後續不可再寫成「外置碎片操作」。
- 魂譜刻印遵守污染值與降頻邏輯，不可無代價連續高強度。
- 信仰品質遵守「強制低質、引導高質」；人類信仰必須來自信任與共同行動，不可硬控。
- 第45章後核心任務是：骨鑰63%補完、守住5%試行配額、準備第一次真實啟取。

### 第46章《補鑰議價》
- 章目標：把「骨鑰補完」從口號變成可執行專案，先拿到第一批補完材料。
- 行動線：
  - 蘇芮拆解盲井九號條件碼，鎖定缺失序列需兩種關鍵材料：校驗鹽 + 回聲台骨纜。
  - 李塵與阿岬截取清洗隊回收車，拿到迦洛私標校驗鹽樣本（只夠做逆向，不夠直接補完）。
  - 灰針線與哨戒鼠線回傳特管組夜班交接節拍，建立「補鑰窗口表」。
- 規則線：
  - 黑潮試圖把第45章拿到的5%試行配額定性為一次性獎勵。
  - 李塵在仲裁補件會提交A級資料貢獻鏈，逼出「可續期七日」附款。
- 可量化收益：
  - 5%試行配額續期 +7日。
  - 骨鑰序列由63%推進到72%（取得可驗證片段，不是猜碼）。
- 可見代價：
  - 李塵為搶截窗口短開高強度節律，裂痕回流上跳一格，出現第二次語義錯置。
- 章尾鉤子：補鑰所需第二批材料位置指向「白燼庫外迴路」，必須在公開監看下動手。

### 第47章《白燼反熔》
- 章目標：在不觸發全面封鎖下，拿到補完骨鑰的核心序列。
- 行動線：
  - 聯航見證隊全程上鏈，李塵小隊突入白燼庫外迴路，不碰主庫只抽更新殘值。
  - 灰醫在T-17架臨時反熔台，將校驗鹽與殘值做逆向對沖。
  - 中段遭教團執行士反包抄，阿岬斷後，隊伍硬撤保住主樣本。
- 規則線：
  - 教團以「封材不可逆」指控序號13竊改封條。
  - 李塵反提三方見證錄像 + 樣本時間戳，將事件定性成「外迴路維護抽樣」。
- 可量化收益：
  - 骨鑰序列由72%推進到88%。
  - 拿到仲裁處中間認證：`補鑰流程具合法驗簽效力`。
- 可見代價：
  - 穩定劑庫存一次消耗40%，灰醫線補給壓力暴增。
  - 阿岬舊傷復發，後續兩章機動能力受限。
- 章尾鉤子：剩餘12%不在教團庫，而在黑潮配額鎖的冷備份裡。

### 第48章《冷備拆鎖》
- 章目標：補齊骨鑰最後12%，拿到第一次真實啟取資格。
- 行動線：
  - 女船長帶中立運隊做外環煙幕，李塵與蘇芮切入配額冷備節點。
  - 以「兩短一長」改編成隊內撤離信號，降低節點內通訊被封時的失聯風險。
  - 清洗隊提前三分鐘進場，李塵只能放棄副目標，保主序列撤離。
- 規則線：
  - 黑潮審務官要求用「監管試入四小時」換取冷備資料合法性。
  - 李塵接受時限但加上條款：若提交A級啟取前置資料，下一輪自動升六小時窗口。
- 可量化收益：
  - 骨鑰序列88% -> 100%（完整可用版）。
  - 拿到一次`六小時啟取演訓窗`（條件性生效）。
- 可見代價：
  - 李塵記憶斷片從「名字錯置」升級到「短時任務步驟遺失」，必須引入副指揮冗餘。
- 章尾鉤子：聯航內部出現洩密跡象，下一章先處理「誰能一起下井」。

### 第49章《聯航護約》
- 章目標：整編聯航，把主角從單隊主力轉成可複製的共同體指揮。
- 行動線：
  - 李塵不先清算內鬼，先跑一趟高風險護航任務，測試聯航新流程是否能抗滲透。
  - 譚七把第44-48章流程拆成「護約班SOP」，實際帶兩支中小運隊跑完外環補給。
  - 途中遭假凍結指令干擾，聯航靠共同驗簽與雙備份證據完成反制。
- 規則線：
  - 聯航啟動續任投票：是否繼續讓序號13擔任驗簽主席。
  - 公會表態「程序中立但支持可預期航線」，關鍵搖擺票倒向李塵。
- 可量化收益：
  - 聯航投票通過（建議寫成11/15），共同驗簽席保住。
  - 5%試行配額延展為15日可用窗口。
- 可見代價：
  - 李塵必須公開部分補鑰流程，失去一段技術壟斷優勢。
  - 人類信任線正式綁到主角身上，失敗成本從「個人輸贏」升級為「整隊沉沒」。
- 信仰升級點：
  - 首次出現人類端「自主追隨」回饋雛形，品質高於鼠群本能信仰但量小。
- 章尾鉤子：六小時演訓窗正式批下，第50章可進行第一次真實啟取。

### 第50章《啟取試刀》
- 章目標：完成第二枚碎片「第一次真實啟取試刀」，驗證整套條件碼可落地。
- 行動線：
  - 李塵、蘇芮、阿岬進井，聯航外圈護航，按三條件碼啟動：第三節點對沖 + 雙節律開鎧 + 完整骨鑰。
  - 啟取中段遭教團封網插入，李塵不硬吞全收益，改採「分段啟取」策略。
  - 成功取出第二枚碎片的外層固定環與共鳴核，確認可在下一輪完成主體剝離。
- 規則線：
  - 教團以「越權取神性核心」申請當場凍結。
  - 聯航見證隊即時上鏈啟取憑證，仲裁端只能先判「條件合規、爭議待審」。
- 可量化收益：
  - 第二枚碎片進度由0%推進到「啟取完成度約40%（固定環+共鳴核到手）」。
  - 因A級成果追加，井口分層貨權試行由5%上調至8%（短期）。
- 可見代價：
  - 李塵裂痕回流觸發強制休整72小時，短期失去高強度輸出資格。
  - 蘇芮雙節律過載，需停用主聽壁儀一章恢復。
- 章尾鉤子：第51章轉入「無主角爆發輸出條件下」的聯航承壓戰，驗證人類信仰線是否真能托底。
